Short-term acceptable result:
Generated static room-cache records are used directly by runtime rendering.
The mutable global cache arrays are removed or bypassed for cooked rooms.

Long-term compatible direction:
Room cache payloads should be streamable chunk assets later.
Do not hard-code an architecture that requires every room cache to stay resident forever.
Represent the cache as per-room/per-chunk slices so a future CD stream pack can page only nearby chunks.
